# Sistema de Gerenciamento de Biblioteca <img src="https://raw.githubusercontent.com/devicons/devicon/master/icons/java/java-original.svg" alt="Java Logo" width="30" height="30" style="vertical-align: middle;">

Sistema de biblioteca feito em Java pra praticar. Permite cadastrar livros, emprestar, devolver e buscar por título. Estou aplicando POO, enum e listas. Projeto em evolução, onde vou adicionando melhorias conforme aprendo mais.

---

## 📌 Funcionalidades

- 📖 Cadastro de livros
- 🔍 Busca de livros por título
- 📦 Listagem de livros disponíveis e emprestados
- 🔄 Empréstimo de livros
- ✅ Devolução de livros

---

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento e organização de classes
- Uso de `enum` para controle de status
- Manipulação de listas (`ArrayList`)
- Estruturas de controle (if, switch, loops)
